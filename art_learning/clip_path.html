Creating Curves with CSS clip-path

How path() works: The path() function uses SVG path syntax:

  • M = Move to (starting point)
  • L = Line to
  • Q = Quadratic Bezier curve (control point + end point)
  • C = Cubic Bezier curve (2 control points + end point)
  • Z = Close path
